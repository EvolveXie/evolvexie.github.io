A flexible view for providing a limited window into a large data set.

RecyclerView 在 support.v7 中提供支持，可以说是 ListView 的升级加强版，建议原有的 ListView 设计的列表都可以转为使用 RecyclerView 实现。

相比 ListView 优点

  • 封装了 ViewHolder，方便开发者快速构建性能良好的列表显示。
  • 把列表的整体布局样式完全交给 LayoutManager 管理，通常只需简单修改 LayoutManager 可以轻松实现纵向列表，横向列表，表格列表以及瀑布流列表。
  • 如果要修改布局样式，只需要重写 layoutManager

RecyclerView 基础使用

使用上和 ListView 基本类似，这里我以纵向列表为例。

  1. 首先在主布局加入 RecycleView

    注意，需要在 build.gradle 中加入依赖(版本自定)

<?xml version=”1.0” encoding=”utf-8”?>

<android.support.v7.widget.RecyclerView
    android:id="@+id/rv_numbers"
    android:layout_width="match_parent"
    android:layout_height="match_parent"/>


public class MyMainActivity extends AppCompatActivity {

private static final int NUM_LIST_ITEMS = 100;
private RecyclerAdapter mAdapter;
private RecyclerView mNumbersList;

@Override
protected void onCreate(Bundle savedInstanceState) {
    super.onCreate(savedInstanceState);
    setContentView(R.layout.activity_main);

    mNumbersList = (RecyclerView) findViewById(R.id.rv_numbers);

    LinearLayoutManager layoutManager = new LinearLayoutManager(this);
    // 设置为纵向线性列表
    mNumbersList.setLayoutManager(layoutManager);
    mNumbersList.setHasFixedSize(true);

    mAdapter = new RecyclerAdapter(NUM_LIST_ITEMS);
    // 设置适配器
    mNumbersList.setAdapter(mAdapter);
}

static class ColorGenerater {
    /**
     * 随机生成随机RGB颜色字符串值
     */
    public static String getRandomColorString() {
        Random random = new Random();
        String r = Integer.toHexString(random.nextInt(256));
        String g = Integer.toHexString(random.nextInt(256));
        String b = Integer.toHexString(random.nextInt(256));
        if (r.length() == 1) {
            r = "0" + r;
        }
        if (g.length() == 1) {
            g = "0" + g;
        }
        if (b.length() == 1) {
            b = "0" + b;
        }
        return "#" + r + g + b;
    }
}

}

<?xml version=”1.0” encoding=”utf-8”?>

<TextView
    android:layout_width="wrap_content"
    android:layout_height="wrap_content"
    android:textSize="14sp"
    android:fontFamily="monospace"
    android:layout_gravity="center_vertical"
    android:gravity="start"
    android:id="@+id/tv_item_number"/>


public class RecyclerAdapter extends RecyclerView.Adapter {

private static final String TAG = RecyclerAdapter.class.getSimpleName();

private int mNumberItems;

public RecyclerAdapter(int numberOfItems) {
    mNumberItems = numberOfItems;
}

@Override
public NumberViewHolder onCreateViewHolder(ViewGroup viewGroup, int viewType) {
    Context context = viewGroup.getContext();
    int layoutIdForListItem = R.layout.number_list_item;
    LayoutInflater inflater = LayoutInflater.from(context);
    boolean shouldAttachToParentImmediately = false;

    View view = inflater.inflate(layoutIdForListItem, viewGroup, shouldAttachToParentImmediately);
    NumberViewHolder viewHolder = new NumberViewHolder(view);

    return viewHolder;
}

@Override
public void onBindViewHolder(NumberViewHolder holder, int position) {
    Log.d(TAG, "#" + position);
    holder.bind(position);
}

@Override
public int getItemCount() {
    return mNumberItems;
}

class NumberViewHolder extends RecyclerView.ViewHolder {
    public NumberViewHolder(View itemView) {
        super(itemView);
        listItemNumberView = (TextView) itemView.findViewById(R.id.tv_item_number);
        mItemContainer = (FrameLayout) itemView.findViewById(R.id.fl_item_container);
    }

    TextView listItemNumberView;
    FrameLayout mItemContainer;

    void bind(int listIndex) {
        String randomColorString = MyMainActivity.ColorGenerater.getRandomColorString();
        listItemNumberView.setText(randomColorString + " " + String.valueOf(listIndex));
        mItemContainer.setBackgroundColor(Color.parseColor(randomColorString));
    }
}

}
`` 在重写方法onCreateViewHolder(ViewGroup viewGroup, int viewType) 中，viewType可以判断视图类型，其值来源于我们重写的int getItemViewType(int position)` 方法，通过这种方式，实现多个 ViewHolder 类，我们可以实现 list 列表项多样的布局方式，比如改变第一项的大小样式等。有兴趣的读者可以尝试，这里暂时不做示例。

总结

以上，可见 RecyclerView 的实现非常简单，和 ListView 的实现方法大同小异。主要是以下几个步骤

  1. 添加依赖
  2. 往布局添加 RecyclerView
  3. 实现 RecyclerViewAdapter
  4. Activity 中实例化并且设置布局
